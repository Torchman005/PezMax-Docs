## v1.1.0 🚀 - 用户体验全面升级与功能增强
*发布日期: 2026-06-09*

PezMax 1.1.0 带来了众多期待已久的功能与改进！从更友好的注册流程、强大的学科搜索，到安装程序的自定义路径支持，我们致力于让您的使用体验更加流畅与个性化。

### ✨ 核心功能与新特性 (Features)
- **学科科目检索**：新增学科目录搜索功能，现在可以快速在搜索页面检索已审核通过的所有学科科目，资源查找效率大幅提升。
- **图片预览重构**：彻底重构了图片类型资料的预览页面，完全符合用户日常的图片查看操作逻辑，带来更直观的视觉交互体验。
- **注册免责声明**：在新用户注册流程中新增了免责声明和温馨提醒，为您的使用体验提供更完善的权益保障与引导。
- **文件大小限制优化**：根据实际使用场景调整并优化了文件大小限制，更贴合学习资料库的实际需求。
- **更新源配置**：完成了应用更新源的配置与对接，为后续的稳定更新与功能迭代奠定了坚实基础。
- **安装路径自定义**：新增安装程序自定义安装路径功能，用户可以自由选择安装位置，充分满足个性化需求。

### 🛠️ Bug 修复 (Fixes)
- **路径显示错误**：修复了部分场景下文件或目录路径显示异常的 bug，确保路径信息准确无误。
- **在线更新修复**：彻底解决了在线更新功能的稳定性问题，让您的版本更新流程更加顺畅。
- **举报页面 UI 修复**：修复了举报页面的 UI 显示错误，视觉呈现更加统一与美观。
- **赞赏页面图标错误**：修复了赞赏页面的图标显示异常问题，界面细节更加精致。

## v1.0.0 🎉 - 拼图满绩 桌面端 正式发布与视觉重构
*发布日期: 2026-05-23*

PezMax-拼图满绩 1.0 正式发布！这是专为学习者打造的新一代桌面端资料库。同时，在此版本中我们对整个应用的视觉语言进行了深度的重构，并在底层渲染管线上做出了极大的性能优化，确保了绝佳的 Apple / Notion 级丝滑体验。

### ✨ 核心功能与新特性 (Features)
- **本地资源管理**：支持连接 RuoYi 后端，自动拉取结构化的树形资源列表（试题、课件、资料等）。
- **多格式文件预览**：右侧集成代码高亮与文本预览器，支持快捷键 `Ctrl + W` 关闭标签页等多个快捷键操作。
- **文件上传与贡献**：支持拖拽上传本地文件至 MinIO 对象存储，并记录贡献者积分。
- **Bento Box 书签管理**：全新的“外部书签”交互逻辑。添加资源表单采用便当盒（Bento）网格布局，支持本地直接选择封面并进行即时的 Base64 本地预览。
- **全局丝滑 Toast 提示**：移除了生硬的原生 Message 弹窗，改为采用带有 Apple 弹性阻尼动画的自定义顶部居中毛玻璃提示。
- **资源树高级操作**：在左侧文件树的“更多”菜单中，新增了“收起所有节点”、“展开所有节点”以及“一键隐藏/显示文件后缀名”的快捷功能。

### 🎨 界面与体验优化 (UI/UX)
- **基础主题系统**：支持浅色/深色主题的无缝切换，并支持简单的自定义纯色背景。
- **悬浮操作“药丸” (Pill Button)**：右下角的“下载原件”和“文件详情”按钮彻底重绘。增加了带有主题色的发光轮廓与内部微弱光晕，并在极度透明的玻璃体内实现了图标的霓虹发光效果。
- **下拉框穿透修复**：彻底解决了 `el-select` 挂载到 body 后，导致的下拉框透明、不可见以及鼠标点击穿透到底层组件的恶性样式问题。
- **详情页布局间隙修复**：调整了书签详情页底部 CSS Grid 布局，自动利用空间并填补了右下角的视觉缺口。
- **按钮与输入框圆角**：统一加大了所有表单输入框和弹窗的圆角（Radius），使其更加现代和柔和。

### 🛠️ 性能与 Bug 修复 (Fixes & Perf)
- **渲染卡顿优化 (Performance)**：移除了设置面板和加载动画中高耗能的动态模糊（`backdrop-filter` 过渡），降低了全局基础模糊半径，并在调整侧边栏宽度时自动开启 GPU 硬件加速并禁用模糊，彻底解决了拖拽和滚动时的掉帧卡顿问题。
- **文件后缀显示异常**：修复了在部分无后缀文件下提取扩展名导致的 `undefined` 错误。
- **封面上传逻辑**：修复了因后端需要 `bookmarkId` 才能构建 MinIO 路径，导致必须先创建书签才能上传封面的问题。现在采用了“先存 MySQL 拿 ID，再延时上传文件并更新 URL”的二步协同逻辑。

### 📦 架构选型
- **底层引擎**：Electron + Node.js 赋能，提供真正的系统级操作权限。
- **视图层**：Vue 3 + Composition API + Vite 构建，极致的开发与编译体验。
- **组件库**：基于 Element-Plus 进行了深度的定制化覆写。
